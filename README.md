# 공유 일기 REST API 📔
- 일기를 공유하고 구독, 좋아요, 댓글로 상호작용하는 어플리케이션
- Springboot/JPA/MySQL + AWS(EC2/RDS/S3)
- 일기 CRUD 기능, 좋아요순/조회순/날짜순 정렬 및 검색 기능
- 일기 공유 기능
- 좋아요, 구독, 북마크 기능
- JWT Token 기반 인증 방식 구현
